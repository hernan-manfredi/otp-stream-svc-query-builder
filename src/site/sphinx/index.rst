####################
Universal Processor
####################

The ``Universal Processor`` is a Quarkus streaming component can apply script-based custom logic to messages.

The scripts can be bundled into the deployment or can be fetched from a remote source.

It makes use of a configurable routing mechanism that enables inbound per-topic message filtering and outbound multi-topic publishing.
